Block-level Elements

A block-level element always start on a new line, and the browsers automatically add some space( a margin) before and after the element
A block-level alement always take up the full width available (stretches out to the left and right as far as it can).
Two commonly used block elements are: p and div

Inline Elements

An inline elemet does not start on a new line.
An inline only takes up as much width as necessary.
