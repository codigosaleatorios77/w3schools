CSS Color Values

In CSS, colors can also be specified using RGB values, HEX values, HSL values, RGBA values, and HSLA values:

Same as color name: Tomato

rgb(255,88,71)

#ff6347

hls(9, 100%,64%)

Same as a color name: Tomato, but 50% transparent:

rgba(255, 99, 71, 0.5)

hsla(9, 100%, 64%, 0.5)
